The Mercurial JavaScript Quiz Board

A javaScript and Bootstrap projects to test a user's knowledge on the javaScript language

Instructions

Presented herein is a series of javaScript Multiple choice questions. Users are to choose the correct answer (or answers) to each question. Users must answer all questions. After completing the test, click on the Submit button at the end of the series. The user will then be able to view their results by clicking the check button on top of their page.
